Open Design for Linux

Unofficial native Linux packaging for Open Design. Install on Ubuntu, Debian, or any Linux distribution.

Quick install:

curl -fsSL https://cuongducle.github.io/open-design-linux/install.sh | sudo bash

Or add the APT repository:

echo "deb [trusted=yes] https://github.com/cuongducle/open-design-linux/releases/latest/download/ ./" | sudo tee /etc/apt/sources.list.d/open-design.list
sudo apt update
sudo apt install open-design

Note: This is an unofficial project. See the GitHub repository for source code, AppImage downloads, and documentation.
